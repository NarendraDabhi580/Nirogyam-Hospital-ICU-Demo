# Nirogyam ICU & Multispeciality Hospital

This is a modern front-end web application built using React, Vite, and Tailwind CSS. The website showcases the hospital's facilities, expert doctors, and specialized services.

## Development

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

## Build for Production

To create an optimized production build, run:
```bash
npm run build
```
This will output statically deployable files in the `dist` directory.
